The main idea of these analyses is that everything a player does changes (on average, in some quantifiable way) his team's expected probability of winning as compared to a statistically average performance, and if you add them all up, you should arrive at a measure for the number of wins each player has contributed, with respect to an average player.  One caveat is that NCAA stats are not adjusted in any way for strength of schedule...so running up the score on non-conference foes will make you look good.
